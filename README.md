# Site Web Serveur Ark'Eons

Ce site web présente les informations de votre serveur Ark'Eons.

## Installation

1. Créez un nouveau dépôt sur GitHub.
2. Ajoutez les fichiers `index.html`, `style.css`, `script.js` et `README.md`.
3. Allez dans les paramètres du dépôt > Pages.
4. Sélectionnez la branche principale et le dossier racine.
5. Votre site sera accessible à l'adresse `https://votre-nom-utilisateur.github.io/nom-du-depot`.

## Personnalisation

- Modifiez les informations dans `index.html` (nom du serveur, IP, etc.).
- Ajustez les styles dans `style.css`.
- Pour un vrai statut du serveur, intégrez une API comme BattleMetrics ou utilisez un script côté serveur.

## Technologies utilisées

- HTML5
- CSS3
- JavaScript (ES6)
